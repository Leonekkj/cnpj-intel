Perguntas frequentes

Os dados são atualizados com que frequência? +
O agente roda continuamente e atualiza os dados a cada 24 horas. Empresas novas aparecem no sistema em até 48h após abertura na Receita Federal.
É legal usar esses dados? +
Sim. Os dados de CNPJ são públicos e fornecidos pela própria Receita Federal. Seguimos a LGPD utilizando apenas dados cadastrais públicos e e-mails corporativos. Não utilizamos dados pessoais sensíveis.
Posso cancelar a qualquer momento? +
Sim, sem multa e sem burocracia. Você acessa o painel, clica em cancelar e o plano é encerrado no fim do período pago.
Como recebo o acesso após o pagamento? +
O acesso é liberado automaticamente em até 5 minutos após a confirmação do pagamento. Você recebe login e senha no e-mail cadastrado.
O plano Enterprise tem API? +
Sim. O plano Enterprise inclui acesso à nossa API REST com documentação completa, permitindo integração com qualquer CRM, planilha ou sistema interno.
Qual a diferença entre os planos? +
A principal diferença é o volume de CNPJs enriquecidos por mês e os tipos de dados disponíveis. O Starter inclui telefone e e-mail. O Pro adiciona Instagram e dados completos dos sócios. O Enterprise adiciona API e volume ilimitado.
